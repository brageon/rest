# Saddle points with threading

Usage: 1) socks, prompt, netcat -l 1234. Since concurrent is not possible because of GIL I deleted km.py which would write to line.csv while prompt is running. Like if it was lazy evaluation. Log would write to lisp.csv which would represent saddles, zero derivatives. 

Bert use *raw* transformers. Ransen use Roberta instead. Lda use glove from Stanford. Parser is standalone file for grammatical correct sentences independent from topic. Tags read {n}a.txt files - which would be chunked bag-of-words on its own. Neow read b.txt files and write to c.txt files. Then wcl read c.txt files to attach its perceptron findings to a.txt files in fixed positions. Tags don´t have perceptron in its setup. Making it random despite ordinals. Purpose of *static* is to sort BOWs on its word class.
