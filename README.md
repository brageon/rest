# RST - heuristics to ranking
This was meant for cognitive therapy by emotional awareness. In its current shape it can be used for creative writing to blogs. Why specific values before measure you might ask. Intuition. Celsius used water as unit and I use political rhetorics from 2018 as unit. Zero product property is not applicable as first order unit. 

Strategies I have tested until glove embeddings. 1) energy_bids.csv to wordlists (dam, dan, eno) from bag-of-words. 2) matrix arithmetic to svm. 3) frequency table to represent 1/6 samples. 4) probability score direct from ranks to trigram distribution: Colin 29 %, Stephen Hicks 34 %, Aborisade Olasunkanmi 41 %. Not consistent with V-L-S. Updated it with descriptive values like "secure accounting score". More guess work for conceptual enhancement (intro) with OODA tempo used in Stratego.
