# Storyboarding 🎬
Blake Snyder: 9-1-2-3-8-7-x-5-6-8-4-5-9 (7th: 9w1)
1. Opening Image: Initial Position (sets the scene)
2. Setup: Iteration (recurring elements and themes)
3. Theme: Antithesis (contrasts the hero's situation with their desire)
4. Catalyst: Direct Proportion. Debate: Terminal Position
5. B-Story: Surprise (a secondary plotline that runs parallel to the main story)
6. Fun and Games: Rhythm (a lighter, more upbeat section of the story)
7. Midpoint: Inverse Proportion (a turning point in the story)
8. Bad Guys Close: Suspense (rising tension)
9. All Is Lost: Terminal Position (a moment of despair or hopelessness)
10. Dark Night of the Soul: Pause (a period of introspection or self-doubt)
11. Break into Three: Inverse Proportion (to suspense)
12. Finale: Terminal Position (the final resolution)
13. Final Image: Initial Position (a return to the opening, often with a new idea)
